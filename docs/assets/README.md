# Motion Screenshots

Place product screenshots here for the README.

## Required files

| Filename | Description | Recommended size |
|----------|-------------|-----------------|
| `screenshot-editor.png` | Main block editor with a document open | 1280×800 |
| `screenshot-ai-chat.png` | AI Chat panel open, showing a conversation | 1280×800 |
| `screenshot-graph.png` | Knowledge graph with linked documents | 1280×800 |
| `screenshot-history.png` | Version history panel showing commits | 1280×800 |

## Tips

- Use dark mode for a more visually striking screenshot
- Use a realistic knowledge base with several linked documents
- For GIFs: record at 1280×800, max 5s, use a tool like Kap or LICEcap
- Compress PNGs with `pngquant --quality=65-80 *.png`
